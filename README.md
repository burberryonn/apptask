# Дейлум

Дейлум — персональное веб-приложение для продуктивности: задачи, привычки и планирование дня на одном экране.

## Текущий формат
Версия работает **без базы данных** и без сервера для хранения данных. Все данные сохраняются локально в браузере через Zustand persist.

## Стек
- Next.js 14 App Router + TypeScript
- Tailwind CSS
- Zustand (локальное хранилище)
- date-fns + Framer Motion
- React Hook Form + Zod (подготовлено для расширения форм)

## Информационная архитектура
- **Главная (`/`)**: лента привычек с круговым прогрессом + группировка задач (Сегодня, Завтра, На этой неделе, Позже), фильтры, быстрое добавление.
- **Календарь (`/calendar`)**: переключение месяц/неделя и просмотр задач/привычек по выбранному дню.
- **Детали привычки (`/habits/[id]`)**: большой прогресс-круг, кнопки +/- и метрики серии.
- **История (`/history`)**: лента событий по задачам и привычкам.
- **Настройки (`/settings`)**: тема, голосовой ввод, заметки по уведомлениям.

## Запуск
```bash
npm install
npm run dev
```

## Модели на клиенте
Используются типизированные сущности:
- задача + подзадачи
- привычка + отметки
- журнал активности
- фильтрация по спискам и скрытым спискам

См. `types/domain.ts` и `lib/store.ts`.

## Архитектурные заметки
- Напоминания: архитектура внутри приложения для одного пользователя.
- Уведомления: Web Notifications как progressive enhancement.
- Голосовой ввод: Web Speech API (`webkitSpeechRecognition`) при поддержке браузера.
- Шаринг: оставлен как аккуратная UI-заглушка на будущее.

## Правовой/дизайн-комментарий
Интерфейс оригинальный и не копирует проприетарные бренды/активы сторонних продуктов.
